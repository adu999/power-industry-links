## 🚀 功能特性

- 🔍 **实时搜索**：支持按名称、简称、网址、描述模糊搜索
- 📂 **分类筛选**：17大分类快速切换
- 📊 **统计面板**：实时显示机构数量
- 🏗️ **组织架构**：两网五大六小展示组织架构信息
- 📈 **上市公司数据**：股票代码 + 最新市值
- 📱 **响应式设计**：完美适配PC/平板/手机
- ⬆️ **回到顶部**：滚动超过一屏自动显示

## 🛠️ 如何维护

### 添加新机构
编辑 `data.js`，在对应分类中添加：
```javascript
{ name: "机构名称", abbr: "简称", url: "https://...", desc: "描述", code: "股票代码", marketCap: "市值" }
```

### 添加新分类
在 `data.js` 中添加新的 section 对象，并在 `app.js` 的 `catLabels` 中添加对应中文标签。

### 修改样式
编辑 `style.css` 中的 CSS 变量即可快速换色：
```css
--primary: #1a5276;      /* 主题色 */
--accent: #e67e22;       /* 强调色 */
--bg: #f4f6f9;           /* 背景色 */
```

## 🌐 部署到 GitHub Pages

### 第一步：创建 GitHub 仓库
```bash
# 在 GitHub 上新建仓库，例如：power-industry-links
# 然后本地初始化并推送
cd power-industry-links
git init
git add .
git commit -m "Initial commit: 电力产业链全景图谱"
git remote add origin https://github.com/你的用户名/power-industry-links.git
git push -u origin main
```

### 第二步：开启 GitHub Pages
1. 进入仓库 → Settings → Pages
2. Source 选择 `main` 分支
3. 点击 Save
4. 等待约1-2分钟，访问 `https://你的用户名.github.io/power-industry-links/`

### 第三步：自定义域名（可选）
1. 在仓库根目录创建 `CNAME` 文件，写入你的域名
2. 在域名服务商处添加 CNAME 记录指向 `你的用户名.github.io`

## 🔄 后续更新维护

### 更新数据
1. 编辑 `data.js` 修改/添加/删除机构信息
2. 提交并推送：
```bash
git add data.js
git commit -m "更新：新增XX机构/更新XX网址"
git push
```
3. GitHub Pages 会自动重新部署（约30秒内生效）

### 更新代码
- 修改 `app.js` → 提交推送 → 自动生效

## 📝 注意事项

1. **网址验证**：添加新网址前请先访问确认有效性
2. **市值更新**：上市公司市值会波动，建议定期更新 `marketCap` 字段
3. **分类准确**：新机构请放到最合适的分类中
4. **描述简洁**：`desc` 字段建议控制在20字以内
5. **无敏感信息**：不要提交任何私密信息或内部链接
